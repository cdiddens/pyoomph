.. _installcompile:

Compilation from source
-----------------------

The following sections discuss the possibility of building pyoomph from source. This is only covered for Mac and Linux, since it is quite complicated to compile all dependencies on Windows. For using the advanced eigensolver SLEPc in Windows, we recommend the installation of pyoomph within WSL, see :numref:`installwsl`.
